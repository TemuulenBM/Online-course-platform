React дээр custom hooks хэрхэн зөв implement хийх вэ?

Ariunbold Bat-Erdene

2 цагийн өмнө • 156 үзсэн

Сайн байцгаана уу? React төсөл дээр ажиллаж байхдаа давтагдаж байгаа логикуудыг useCustomHook болгож салгах гэж оролдож байна.

Миний асуулт бол, custom hook дотор өөр hooks (жишээ нь useEffect, useState) ашиглах нь зөв үү? Мөн clean-up функцүүдийг хэрхэн зохион байгуулбал хамгийн тохиромжтой вэ?

Жишээ код:

function useWindowSize() {
  const [size, setSize] = useState(window.innerWidth);

  useEffect(() => {
    const handleResize = () => setSize(window.innerWidth);
    window.addEventListener('resize', handleResize);
    return () => window.removeEventListener('resize', handleResize);
  }, []);

  return size;
}

Энэ арга зөв үү эсвэл илүү оновчтой шийдэл бий юу?

Тийм ээ, таны жишээ код маш зөв байна. Custom hook-ийн гол зорилго нь React-ийн build-in hook-уудыг ашиглан логикийг encapsulate хийхэд оршдог. Таны useEffect доторх clean-up (window.removeEventListener) нь memory leak-ээс сэргийлэх маш чухал хэсэг юм.

Solongo P. 45 минутын өмнө

Нэмж хэлэхэд, хэрвээ та hook-ээ олон газар ашиглахаар бол debounce хийж өгвөл гүйцэтгэлд (performance) сайнаар нөлөөлнө шүү.
